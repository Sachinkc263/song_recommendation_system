# Spotify Music Recommendation System

A **content-based song recommender** built on the Spotify Music Recommendation dataset from Kaggle. Given any song in the dataset, it returns the N most musically similar tracks using cosine similarity over a 14-dimensional audio feature vector.

## Datasets

All datasets come from the [Spotify Music Recommendation System](https://www.kaggle.com/) Kaggle dataset.

| File | Rows | Description |
|---|---|---|
| `data.csv` | 170,653 | Individual tracks with full audio features (1921–2020) |
| `data_w_genres.csv` | 28,680 | Artist-level aggregated features + genre tags |
| `data_by_artist.csv` | 28,680 | Mean audio features per artist |
| `data_by_genres.csv` | 2,973 | Mean audio features per genre |
| `data_by_year.csv` | 100 | Mean audio features per year |
| `song_and_artists.csv` | 2,420 | Curated song list with user ratings |

## Feature Vector (14 dimensions)

| Feature | Transform |
|---|---|
| `valence`, `acousticness`, `danceability`, `energy`, `loudness` | MinMax → [0, 1] |
| `instrumentalness`, `speechiness`, `liveness` | log1p + MinMax → [0, 1] |
| `tempo` | StandardScaler + MinMax → [0, 1] |
| `key` | Cyclic encoding (sin + cos) → [−1, 1] |
| `mode`, `explicit` | Binary (0 / 1) |
| `popularity` | MinMax → [0, 1] |

## Quick Start

### Local (with pip)

```bash
# 1. Clone and set up environment
python -m venv .venv
.venv\Scripts\activate          # Windows
pip install -r requirements.txt

# 2. Place raw CSVs in data/raw/

# 3. Launch Jupyter
jupyter notebook
```

### Docker

```bash
cd docker
docker-compose up --build
# Open http://localhost:8888
```

## Usage (Python)

```python
import pickle, numpy as np
from src.models.recommender import ContentBasedRecommender

with open("models/recommender_payload.pkl", "rb") as f:
    payload = pickle.load(f)

rec = ContentBasedRecommender(
    feature_matrix=payload["feature_matrix"],
    track_index=payload["track_index"],
)

print(rec.recommend("Bohemian Rhapsody", top_n=10))
```
